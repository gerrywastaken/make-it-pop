# Make It Pop - UI/UX Mockups

This directory contains interactive HTML mockups demonstrating proposed UI/UX improvements for the Make It Pop extension.

## Mockups

### 1. `tabbed-settings.html`
**Concept**: Tabbed interface for settings page
- Separates Groups, Domains, and Import/Export into tabs
- Reduces vertical scrolling
- Better organization and mental model

### 2. `enhanced-popup.html`
**Concept**: Improved popup with context awareness
- Shows current domain
- Displays active groups for the current page
- Quick group toggles without opening settings
- Stats and indicators

### 3. `inline-editing.html`
**Concept**: Inline editing for groups
- Edit groups directly in their cards
- No separate form area
- Phrase tags/chips instead of textarea
- Faster workflow

### 4. `mobile-responsive.html`
**Concept**: Mobile-optimized settings page
- Responsive layout with breakpoints
- Touch-friendly controls (larger targets)
- Bottom sheet for actions
- Collapsible sections

### 5. `color-presets.html`
**Concept**: Color preset picker
- Pre-defined color schemes for common use cases
- Visual swatches
- One-click application

### 6. `empty-states.html`
**Concept**: Improved empty states and onboarding
- Helpful messaging when no groups/domains exist
- Guided first-time experience
- Visual illustrations

## How to View

Open any HTML file in a web browser to see the interactive mockup. These are standalone demonstrations and don't connect to the actual extension code.

See each mockup for notes on implementation complexity and recommended priority.
